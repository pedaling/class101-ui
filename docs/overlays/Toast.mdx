# Toast

오류, 알림 등을 알리기에 유용한 컴포넌트입니다. `Toaster.create`를 사용합니다.

## Props

<PropsTable of={Toast} />

## Basic usage

```tsx
let AppToaster;

export async function showToast(props: ToastProps) {
  if (!AppToaster) {
    AppToaster = await Toaster.create();
  }
  AppToaster.show(props);
}
```

<Playground>
