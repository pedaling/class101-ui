# Button

눌러서 제어하는 기능을 담당하는 컴포넌트입니다.

> **NOTE**
>
> - v4부터 Icon 컴포넌트와 함께 사용할 수 있도록 flex 기반의 스타일링을 지원하고 있습니다.
> - v4부터 textAlign 프로퍼티는 지원하지 않습니다.
> - v6부터 src를 지원하지 않습니다. 버튼의 아이콘은 Icon 컴포넌트를 사용해주세요.
> - v7부터 기존의 color, backgroundColor는 지원하지 않습니다.
> - v8부터 href과 target 프로퍼티는 사용하지 않을 예정입니다.
