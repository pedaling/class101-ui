## With Image Ratio

```ts
export enum CoverRatio {
  RATIO_16X9 = 0.5625,
  RATIO_16X10 = 0.625,
  RATIO_4X3 = 0.75,
  RATIO_3X4 = 1.333,
  RATIO_1X1 = 1,
}
```

<Playground>
  <Grid>
    <Row>
