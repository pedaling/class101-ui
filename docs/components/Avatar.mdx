import { Playground, Props } from 'docz';
import { Avatar, AvatarSize, Icon, Colors, Badge } from '@class101/ui';
import { IMAGE_URL } from './Avatar';

# Avatar

Avatars can be used to represent people profile.
It can be used with a badge, and can display a number of notifications or a specific icon.

## Props

<Props of={Avatar} />

## Basic usage

src가 없거나 이미지 로딩에 실패했을 때 text가 보이게 됩니다.

<Playground>
  <Avatar src={IMAGE_URL} />
  <Avatar src="invaild_url" text="가나다라" />
  <Avatar src={IMAGE_URL} icon={<Icon.Heart fillColor={Colors.orange600} />} />
</Playground>

## With Icon

Avatar의 icon에 들어가는 컴포넌트는 자동적으로 size={number}를 주입받습니다.

`<Badge />`나 `<Icon />` 컴포넌트를 넣는 것을 추천합니다.

### Badge

<Playground>
  {() => {
    const badgeWithIcon = (
      <Badge backgroundColor={Colors.orange600} pill>
        <Icon.Crown fillColor={Colors.white} />
      </Badge>
    );
    return <Avatar size={AvatarSize.LARGE} src={IMAGE_URL} icon={badgeWithIcon} />;
  }}
</Playground>

## Event

`onClick`과 `onIconClick`을 둘 다 설정한 경우 `onIconClick`이 실행된 후 `onClick`이 실행됩니다.

원하지 않는 경우 `e.stopPropagation()`를 실행해서 전파를 막아야 합니다.

<Playground>
  <Avatar
    src={IMAGE_URL}
    icon={<Icon.Heart fillColor={Colors.orange600} />}
    onClick={() => alert('click1')}
    onIconClick={() => alert('click2')}
  />
  <Avatar
    src={IMAGE_URL}
    icon={<Icon.Heart fillColor={Colors.orange600} />}
    onClick={() => alert('click1')}
    onIconClick={e => {
      alert('click2');
      e.stopPropagation();
    }}
  />
</Playground>
