# Typography

Class101 UI의 Typography는 기기나 뷰포트 크기가 증가함에 따라 적절하게 확대되는 반응형, 모바일 우선 유동 글자 시스템을 지원합니다.

## Basic usage

<Playground>
  <Headline1>H1 heading</Headline1>
  <Headline2>H2 heading</Headline2>
  <Headline3>H3 heading</Headline3>
  <Subtitle1>H4 subtitle</Subtitle1>
  <Body1>P and div body1</Body1>
  <Body2>P and div body2</Body2>
  <Caption1>Div caption1</Caption1>
  <Caption2>Div caption2</Caption2>
</Playground>

## Responsive examples

lg: >= 1024px <br />
sm: < 1013px

데스크탑(lg)과 모바일(sm)에서 태그와 스타일이 바뀝니다! 브라우저 창의 가로를 늘렸다 줄였다 해보세요!

<Playground>
  <Headline1 sm="headline2">H1 heading or H2 heading</Headline1>
  <Body1 sm="Body2">DIV body1 or DIV body2</Body1>
  <Caption1 sm="Caption2">DIV caption1 or DIV caption2</Caption1>
</Playground>
