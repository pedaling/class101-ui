# Icon

Icons are created in SVG, and you can easily apply colors and effects through SVG or CSS properties.

## Import

```
import { Icon } from '@class101/ui';
```

## Props

<PropsTable of={Icon} />

## Basic usage

사용할 수 있는 아이콘은 제플린 파일을 참고하세요. 제플린의 `ic-checkbox-on` 컴포넌트의 경우 `<Icon.CheckboxOn />`와 대응합니다.

<Playground>
  <Add />
  <Icon.BellOutline />
  <Icon.Alert />
  <Icon.CheckboxOn />
  <Icon.Close />
</Playground>

## Size

Change the size of an icon with the size prop. Icon sizes uses the pixel scale.

<Playground>
  <Icon.BellOutline size={16} />
  <Icon.BellOutline size={24} />
  <Icon.BellOutline size={28} />
  <Icon.BellOutline size={36} />
  <Icon.BellOutline size={42} />
  <Icon.BellOutline size={48} />
</Playground>

## Color

You can set up to 2 colors for icons.
It is divided into basic fill color and accent color. The `accentColor` is only a specific icon.
The props that set the basic fill color are `fillColor`, which is present on all icons.

### Fill color

<Playground>
  <Icon.BellOutline fillColor={Colors.gray700} />
  <Icon.BellOutline fillColor={Colors.red700} />
  <Icon.BellOutline fillColor={Colors.orange700} />
</Playground>

### Accent color

<Playground>
  <Icon.StarHalf />
  <Icon.StarHalf fillColor={Colors.red600} accentColor={Colors.red200} />
  <Icon.StarNone />
  <Icon.StarNone accentColor={Colors.red200} />
</Playground>
