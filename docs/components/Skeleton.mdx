# Skeleton

<PropsTable of={Skeleton} />

## Basic usage

<Playground>
  <Skeleton count={10} />
  <br />
  <Skeleton count={20} color={Colors.orange500} />
  <br />
  <Skeleton count={30} rounded={false} />
  <br />
  <Skeleton count={40} circle />
</Playground>

**count**가 있다면 `display: inline-block`으로 변합니다.
