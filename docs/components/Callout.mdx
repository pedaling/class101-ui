## Action

Callout의 내용과 관련된 액션을 처리하는 버튼입니다.

> 폼과 관련이 없도록 사용해주세요. (submit x)

```ts
interface ActionProps {
  content: string;
  leftIcon?: ReactNode;
  rightIcon?: ReactNode;
  loading?: boolean;
  to?: string;
  onClick?: (event: React.MouseEvent<HTMLButtonElement> | React.MouseEvent<HTMLAnchorElement>) => void;
  disabled?: boolean;
  className?: string;
}
```
