# Layout

Class101 UI는 기기나 뷰포트 크기가 증가함에 따라 12열이 적절하게 확대되는 반응형, 모바일 우선 유동 그리드 시스템을 지원합니다.

> v4 부터는 flex 기반의 레이아웃을 제공합니다.

## Grid options

lg: >= 1024px <br />
sm: < 1013px

## Mobile first

mobile first로 설계되어 sm만 설정 되어 있을 경우 sm에서의 설정이 데스크탑(lg)에서도 적용됩니다.
반대로 lg만 설정할 경우 모바일(sm)에서는 한 셀이 12열을 모두 차지하게 됩니다.

<Playground>
  <Row>
    <Col sm={4}>sm-4</Col>
  </Row>
  <Row>
    <Col lg={4}>lg-4</Col>
  </Row>
</Playground>

## Margin top, bottom

기본 단위는 `px`입니다. `em`, `rem` 등의 다른 단위는 문자열로 넣어주세요.

<Playground>
  <Grid>
    <Row>
      <Col lg={12}>With grid container</Col>
    </Row>
  </Grid>
  <Grid marginTop={24}>
    <Row>
      <Col lg={12}>With grid container - marginTop 24(px)</Col>
    </Row>
  </Grid>
  <Grid marginTop={'2em'} marginBottom={'2em'}>
    <Row>
      <Col lg={12}>With grid container - marginTop 2em, marginBottom 2em</Col>
    </Row>
  </Grid>
  <Grid>
    <Row>
      <Col lg={12}>With grid container</Col>
    </Row>
  </Grid>
</Playground>
