# Typography

> **NOTE**
>
> - v3부터 Bodies에 `paragaph` 속성을 지원하지 않습니다.
> - v6부터는 margin 스타일을 제어하는 속성을 따로 지원하지 않습니다.

Class101 UI의 Typography는 기기나 뷰포트 크기가 증가함에 따라 적절하게 확대되는 반응형, 모바일 우선 유동 글자 시스템을 지원합니다.

## Basic usage

<Playground>
  <Headline1>H1 heading</Headline1>
  <Headline2>H2 heading</Headline2>
  <Headline3>H3 heading</Headline3>
  <Subtitle1>H4 subtitle1</Subtitle1>
  <Subtitle1>H4 subtitle2</Subtitle1>
  <Body1>P and div body1</Body1>
  <Body2>P and div body2</Body2>
  <Caption1>Div caption1</Caption1>
  <Caption2>Div caption2</Caption2>
</Playground>

## element

`element`를 사용해서 다음의 태그들을 적용할 수 있습니다.

```
element: 'h1' | 'h2' | 'h3' | 'h4' | 'h5' | 'h6' | 'p' | 'div';
```

<Playground>
  <Headline1 element="h3">H1 heading or H2 heading</Headline1>
  <Subtitle1 element="h1">H1 heading or H2 heading</Subtitle1>
</Playground>

## Responsive examples

lg: >= 1024px <br />
sm: < 1023px

데스크탑(lg)과 모바일(sm)에서 스타일이 바뀝니다! 브라우저 창의 가로를 늘렸다 줄였다 해보세요!<br/>

```
'Headline1' | 'Headline2' | 'Headline1' | 'Subtitle1' | 'Body1' | 'Body2' | 'Caption1' | 'Caption2'
```

<Playground>
  <Headline1 sm="Headline2">H1 heading or H2 heading</Headline1>
  <Body1 sm="Body2">DIV body1 or DIV body2</Body1>
  <Caption1 sm="Caption2">DIV caption1 or DIV caption2</Caption1>
</Playground>

## Headings

Headings는 더 큰 텍스트 스타일인 `display: 2 | 3`를 지원합니다.

<Props of={Headline1} />

<Playground>
  <Headline1 display={2}>H1 display2</Headline1>
  <Headline1 display={3}>H1 display3</Headline1>
  <Headline1>H1 heading</Headline1>
  <Headline2>H2 heading</Headline2>
  <Headline3>H3 heading</Headline3>
</Playground>
