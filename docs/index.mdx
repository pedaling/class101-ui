# Class101 UI

Class101 UI는 리액트 프론트엔드 개발을 더 빠르고 쉽게 만들어줍니다. 또한 클래스101 프로젝트들을 위해 만들어졌습니다.

## Quick Start

### Install

```console
npm install --save @class101/ui
```

### Usage

> **NOTE**
>
> - v6부터는 HTMLAttributes 속성들은 {HTMLElement}Attributes 속성을 통해 전달합니다.

```jsx
import React, { Component } from 'react';

import { Button } from '@class101/ui';

class Example extends Component {
  render() {
    return <Button>Hello, world!</Button>;
  }
}
```
